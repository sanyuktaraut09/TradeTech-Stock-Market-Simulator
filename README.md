# TradeTech-Stock-Market-Simulator
A virtual stock trading platform that uses Yahoo Finance APIs to simulate real-time stock investments with portfolio management and profit/loss analytics.
📈 TradeTech – Stock Market Simulator
Overview

TradeTech is a virtual stock trading platform that enables users to simulate real-world stock investments without financial risk. The application fetches real-time market data using Yahoo Finance APIs and allows users to buy, sell, and monitor stocks through a virtual portfolio.

The platform provides real-time profit/loss calculations, portfolio tracking, and investment performance analytics, making it an excellent educational tool for learning stock market fundamentals.

Features
📊 Real-time stock prices using Yahoo Finance API
💰 Virtual trading with simulated balance
📈 Live portfolio tracking
📉 Profit & Loss calculation
📋 Transaction history
🔍 Search stocks by ticker symbol
⚡ Fast and responsive interface
📱 Simple dashboard for monitoring investments
